# Xstring

Ruby String extensions.

Currently includes one amazing to_class extension.

## Installation

Add this line to your application's Gemfile:

    gem 'xstring'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install xstring

## Usage

"Object".to_class 			# => Object
"marshmallow".to_class	# => nil
